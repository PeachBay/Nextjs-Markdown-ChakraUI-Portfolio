## Getting Started

To get a local copy up and running, follow these two simple  steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/PeachBay/Nextjs-Markdown-ChakraUI-Portfolio
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the local server
   ```sh
   npm run dev
   ```
